# Retail Sales Data Analysis (R)

## Overview
This project analyzes a transactional retail dataset using R to explore sales performance, customer demographics, and product trends. The goal was to practice data cleaning, statistical analysis, and data visualization techniques commonly used in business analytics.

## Key Tasks Performed
- Cleaned and transformed raw sales data for analysis  
- Converted revenue values to numeric format for calculations  
- Created subsets to analyze sales by state and customer characteristics  
- Generated statistical summaries and frequency tables  
- Built visualizations including histograms and scatter plots  
- Evaluated relationships between variables (e.g., Units Sold vs Revenue)

## Skills Demonstrated
- Data cleaning and preprocessing  
- Statistical analysis in R  
- Data visualization  
- Business data interpretation  
- Logical data filtering and subsetting  

## Tools Used
- R Programming  
- Base R functions for analysis and visualization  

This project demonstrates foundational data analytics skills applied to a business-style dataset.
